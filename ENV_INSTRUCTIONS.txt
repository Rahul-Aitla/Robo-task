# Instructions for .env.local

Add this single line to your .env.local file:

GOOGLE_GENERATIVE_AI_API_KEY=your_gemini_api_key_here

# How to get your Gemini API key:
# 1. Visit: https://aistudio.google.com/app/apikey
# 2. Sign in with your Google account
# 3. Click "Create API key"
# 4. Copy the key and paste it above (replacing "your_gemini_api_key_here")
# 5. Save this file
# 6. The dev server will automatically reload

# Note: Gemini has a generous free tier!
